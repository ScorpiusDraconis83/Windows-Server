
# 🧑‍💼 Windows 11 Enterprise – User Group Capabilities 

Windows uses **user groups** to control what users can and cannot do. Each group has specific powers. Here's a simple guide for beginners:

---

## 📋 User Groups and Their Abilities

| **Group Name**                  | ✅ **What They Can Do**                                                                                  | ❌ **What They Cannot Do**                                                 |
| ------------------------------- | ------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| **1. Administrators**           | - Full control <br> - Change any setting <br> - Install/remove software <br> - Add/remove user accounts | Nothing — they have full power                                            |
| **2. Backup Operators**         | - Backup and restore files <br> - Can read any file during backup, even without permission              | - Cannot change system settings <br> - Cannot read user files normally    |
| **3. Guests**                   | - Use internet <br> - Run installed apps                                                                | - Cannot install apps <br> - Cannot change settings <br> - No file access |
| **4. Power Users**              | - Do some admin work <br> - Share files <br> - Change some settings                                     | - Cannot fully control system <br> - Cannot make major changes            |
| **5. Remote Desktop Users**     | - Log in using Remote Desktop <br> - Use files/apps remotely                                            | - Cannot change settings <br> - No access to other users’ files           |
| **6. Hyper-V Administrators**   | - Create and manage virtual machines (VMs) <br> - Set up virtual networks                               | - Cannot change system-wide settings unless given permission              |
| **7. Network Config Operators** | - Change network settings <br> - Manage Wi-Fi/Ethernet connections                                      | - Cannot change security settings <br> - Cannot access private files      |
| **8. Event Log Readers**        | - View system logs <br> - Check for system issues                                                       | - Cannot edit logs <br> - Cannot access user files                        |
| **9. Crypto Operators**         | - Use encryption <br> - Manage security certificates                                                    | - Cannot change important settings <br> - No access to other files        |
| **10. Performance Log Users**   | - Watch system performance <br> - Collect system speed/data reports                                     | - Cannot change settings <br> - Cannot see other users’ data              |

---

## ✅ Summary

Windows 11 Enterprise gives you different **user groups** to keep things **organized, secure, and manageable**. Each group has a **clear role**—so you can give users just the right amount of access without giving too much control.
