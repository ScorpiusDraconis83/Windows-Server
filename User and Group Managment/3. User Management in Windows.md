
# 👥 User Accounts in Windows 11 Enterprise 

Windows 11 Enterprise has different types of user accounts to control what people can do on the computer.

---

## Types of User Accounts

| **Account Type**     | **What It Means**                                                                              |
| -------------------- | ---------------------------------------------------------------------------------------------- |
| **1. Administrator** | Has full control: can install software, change settings, and manage other users.               |
| **2. Standard User** | Can use apps and change basic settings but cannot affect other users or system security.       |
| **3. Guest Account** | Temporary user with very limited access—good for someone who only needs to use the PC briefly. |

---

## How to Create User Accounts

| **Method**                    | **How to Do It**                                                                                                                              |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. Settings**               | Open *Settings* > *Accounts* > *Family & other users*, then click **Add someone else to this PC**. You can make a local or Microsoft account. |
| **2. Control Panel**          | Use *Control Panel* > *User Accounts* to add or manage users.                                                                                 |
| **3. Local Users and Groups** | Open `lusrmgr.msc` (only in Pro/Enterprise versions) to see detailed user and group controls.                                                 |

---

## Managing User Accounts

| **What You Can Do**            | **Where/How**                                                                                |
| ------------------------------ | -------------------------------------------------------------------------------------------- |
| **Change Account Type**        | Switch a user between Standard and Administrator in *Settings* > *Family & other users*.     |
| **Set or Reset Passwords**     | Admins can change passwords for other users.                                                 |
| **Disable or Enable Accounts** | Temporarily turn off a user account without deleting it, useful for short-term restrictions. |

---

## User Permissions (What Users Can Access)

| **Permission**                | **Meaning**                                           |
| ----------------------------- | ----------------------------------------------------- |
| **Read/Write Permissions**    | Controls if users can open or edit files and folders. |
| **Administrator Permissions** | Only Admins can make big changes to the system.       |

---

## Using Local Users and Groups Tool

| **Feature**         | **How to Use**                                                          |
| ------------------- | ----------------------------------------------------------------------- |
| **Open Tool**       | Press `Windows + R`, type `lusrmgr.msc`, and press Enter.               |
| **View Accounts**   | See all users and groups, including hidden or disabled accounts.        |
| **Manage Accounts** | Change password settings, disable accounts, or change group membership. |

---

## Common User Groups and Their Purpose

| **Group Name**     | **What They Do**                      |
| ------------------ | ------------------------------------- |
| **Administrators** | Full control over the computer.       |
| **Power Users**    | Can do some admin tasks, but limited. |
| **Guests**         | Temporary users with limited access.  |

---

## Useful Commands for User Management

| **Command**      | **What It Does**                           |
| ---------------- | ------------------------------------------ |
| `net user`       | Shows all user accounts on the computer.   |
| `whoami /groups` | Lists groups your current user belongs to. |

---

## ✅ Summary

Managing user accounts well keeps your Windows 11 Enterprise PC **safe** and **organized**. You can create, change, or remove accounts using Settings, Control Panel, or the Local Users and Groups tool, depending on what you need.
