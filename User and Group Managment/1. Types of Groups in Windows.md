
## 🧑‍💻 Local Groups

| Group                | Can Do                                                                                                                          | Cannot Do                                                               |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **Administrators**   | - Modify all system settings  <br> - Create user accounts <br> - Access all files and folders <br> - Install/uninstall software | Nothing — full control                                                  |
| **Backup Operators** | - Backup and restore all files                                                                                                  | - Cannot change system settings <br> - Cannot access other users' files |
| **Power Users**      | - Perform limited admin tasks <br> - Share files                                                                                | - Cannot change important system settings <br> - No full admin rights   |
| **Users**            | - Use standard files and apps                                                                                                   | - Cannot install software <br> - Cannot change settings                 |
| **Guests**           | - Browse the internet                                                                                                           | - Cannot edit files <br> - Cannot change system settings                |

---

## 🔐 Built-in Security Groups

| Group                 | Can Do                      | Cannot Do                           |
| --------------------- | --------------------------- | ----------------------------------- |
| **Administrators**    | All administrative tasks    | Nothing                             |
| **Backup Operators**  | Perform backup operations   | Cannot modify system settings       |
| **Account Operators** | Manage user accounts        | No full administrative rights       |
| **Print Operators**   | Manage printers             | Cannot change system configurations |
| **Server Operators**  | Manage server-related tasks | Cannot access other users' files    |

---

## 🆔 Special Identity Groups

| Group                   | Can Do              | Cannot Do                   |
| ----------------------- | ------------------- | --------------------------- |
| **Authenticated Users** | All logged-in users | No special admin privileges |
| **Everyone**            | Includes all users  | Limited access only         |

---

## 🌐 Domain Groups (Active Directory)

| Group                 | Can Do                           | Cannot Do                              |
| --------------------- | -------------------------------- | -------------------------------------- |
| **Domain Admins**     | Admin tasks across entire domain | May have restrictions on special tasks |
| **Account Operators** | Manage user accounts             | No full admin rights                   |
| **Server Operators**  | Manage servers                   | Cannot access other users’ files       |

---

## 🛠️ Custom Groups

| Group                | Can Do                                   | Cannot Do                              |
| -------------------- | ---------------------------------------- | -------------------------------------- |
| **Marketing Team**   | Marketing tasks, access related files    | Cannot access other departments’ files |
| **Development Team** | Perform development tasks, use resources | Cannot perform administrative actions  |
