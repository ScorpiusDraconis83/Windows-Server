
# 🧑‍💻 Windows User Groups 

In Windows, **user groups** help manage what users can and cannot do. Instead of giving permission one-by-one, users are added to groups, and permissions are handled by the group.

---

## 🔸 **Local Groups**

These groups exist on a single computer (not on a network domain).

| **Group**            | ✅ **What They Can Do**                                                       | ❌ **What They Cannot Do**                                                     |
| -------------------- | ---------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| **Administrators**   | - Change any system setting  <br> - Install programs <br> - Add/remove users | Nothing — they have full control                                              |
| **Backup Operators** | - Backup and restore files                                                   | - Cannot change system settings <br> - Cannot open other users’ private files |
| **Power Users**      | - Do limited admin work <br> - Share files locally                           | - Cannot make big system changes <br> - No full admin rights                  |
| **Users**            | - Use apps and files normally                                                | - Cannot install software <br> - Cannot change important settings             |
| **Guests**           | - Use basic features <br> - Browse internet                                  | - Cannot save changes <br> - Cannot install software or change settings       |

---

## 🔐 **Built-in Security Groups**

These groups are used for system and network security.

| **Group**             | ✅ **What They Can Do**         | ❌ **What They Cannot Do**           |
| --------------------- | ------------------------------ | ----------------------------------- |
| **Administrators**    | All admin tasks                | Nothing                             |
| **Backup Operators**  | Backup system files            | Cannot change system settings       |
| **Account Operators** | Add/edit user accounts         | Cannot make system-wide changes     |
| **Print Operators**   | Manage printers and print jobs | Cannot change system configurations |
| **Server Operators**  | Manage basic server tasks      | Cannot view private user data       |

---

## 🆔 **Special Identity Groups**

These groups automatically include users based on how they log in.

| **Group**               | ✅ **What They Can Do**       | ❌ **What They Cannot Do**    |
| ----------------------- | ---------------------------- | ---------------------------- |
| **Authenticated Users** | Includes all logged-in users | No admin rights              |
| **Everyone**            | Includes all users           | Can only access public areas |

---

## 🌐 **Domain Groups (Used in Active Directory Networks)**

These groups are used in networked environments like offices and schools.

| **Group**             | ✅ **What They Can Do**                | ❌ **What They Cannot Do**             |
| --------------------- | ------------------------------------- | ------------------------------------- |
| **Domain Admins**     | Full control over the whole network   | May be restricted by network policies |
| **Account Operators** | Manage domain user accounts           | Cannot manage entire system settings  |
| **Server Operators**  | Maintain servers and restart services | Cannot access private user files      |

---

## 🛠️ **Custom Groups**

You can create your own groups for specific teams or departments.

| **Group**            | ✅ **What They Can Do**                         | ❌ **What They Cannot Do**                          |
| -------------------- | ---------------------------------------------- | -------------------------------------------------- |
| **Marketing Team**   | Work on marketing tasks, access marketing data | Cannot access development or admin data            |
| **Development Team** | Access development tools and servers           | Cannot manage system settings or other departments |
