
# 🧑‍💼 Windows 11 Enterprise - User Group Capabilities Overview

| **Group Name**                         | **Can Do**                                                                                                                   | **Cannot Do**                                                                                 |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| **1. Administrators**                  | - Change all system settings<br>- Create new user accounts<br>- Access all files and folders<br>- Install/uninstall software | Nothing – full control granted                                                                |
| **2. Backup Operators**                | - Back up and restore all files<br>- Can access files during backup even without permissions                                 | - Cannot change system settings<br>- Cannot access other users' files                         |
| **3. Guests**                          | - Browse the internet<br>- Run installed applications                                                                        | - Cannot change system settings<br>- Cannot access others’ files<br>- Cannot install software |
| **4. Power Users**                     | - Perform some administrative tasks (e.g., file sharing)<br>- Modify limited settings                                        | - Cannot perform full admin tasks<br>- Cannot change critical settings                        |
| **5. Remote Desktop Users**            | - Log in via Remote Desktop<br>- Access files and applications remotely                                                      | - Cannot change system settings<br>- Cannot access other users’ files                         |
| **6. Hyper-V Administrators**          | - Create/manage VMs in Hyper-V<br>- Configure virtual networks and storage                                                   | - Cannot change system settings without required permissions                                  |
| **7. Network Configuration Operators** | - Configure network settings<br>- Manage network connections                                                                 | - Cannot access other users' files<br>- Cannot change security settings                       |
| **8. Event Log Readers**               | - Read Windows event logs<br>- Monitor system activities                                                                     | - Cannot modify logs<br>- Cannot access other users’ files                                    |
| **9. Cryptographic Operators**         | - Encrypt/decrypt data<br>- Manage security certificates                                                                     | - Cannot change system settings<br>- Cannot access others’ files                              |
| **10. Performance Log/Monitor Users**  | - Monitor system performance<br>- Collect performance data                                                                   | - Cannot change system settings<br>- Cannot access other users’ files                         |

---

### ✅ **Conclusion**

Windows 11 Enterprise provides structured user groups with defined capabilities and limitations. These groups enable administrators to manage access, functionality, and security more effectively in an organizational environment.
