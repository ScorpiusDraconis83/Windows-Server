
### How to Install Windows Server 2012 R2 in VirtualBox

#### Requirements

1. **VirtualBox**: Install VirtualBox from [here](https://www.virtualbox.org/).
2. **Windows Server 2012 R2 ISO**: Download it from Microsoft's website if you have a valid license.

### Step 1: Open VirtualBox

* Open VirtualBox and click **New**.

### Step 2: Create a Virtual Machine

1. **Name and Type**:

   * Name: Enter "Windows Server 2012 R2".
   * Type: Select "Microsoft Windows".
   * Version: Choose "Windows 2012 (64-bit)" (or 32-bit if you have a 32-bit ISO).

2. **Allocate RAM**:

   * Set at least 2048 MB (2 GB) of RAM (you can increase it if you have more RAM).

3. **Create Virtual Hard Disk**:

   * Select "Create a virtual hard disk now" and click **Create**.
   * Choose "VDI (VirtualBox Disk Image)" and click **Next**.
   * Choose "Dynamically allocated" and click **Next**.
   * Set the size to at least 20 GB and click **Create**.

### Step 3: Add ISO File

1. **Settings**:

   * Right-click the virtual machine and select **Settings**.
2. **Storage**:

   * In the **Storage** tab, click on **Empty** under "Controller: IDE".
   * Click the CD icon on the right, then choose **Choose a disk file**.
   * Select your Windows Server 2012 R2 ISO file and click **OK**.

### Step 4: Start the Virtual Machine

* Select your Virtual Machine and click **Start**.

### Step 5: Install Windows Server

1. **Start Installation**:

   * Click **Install now** when the setup starts.

2. **License Agreement**:

   * Check "I accept the license terms" and click **Next**.

3. **Install Type**:

   * Select **Custom: Install Windows only (advanced)**.

4. **Select Drive**:

   * Choose the drive where you want to install Windows Server (format it if needed).

5. **Installation**:

   * The installation will start. It may take some time.

### Step 6: Complete the Setup

1. **Initial Setup**:

   * Set your server's **name** and **administrator password**.

2. **Date and Time**:

   * Set the correct time and date.

3. **Network Settings**:

   * Configure network settings as needed.

### Step 7: Install VirtualBox Guest Additions (Optional)

* With your virtual machine running, go to **Devices** in the VirtualBox menu.
* Click **Insert Guest Additions CD image** and install it.
