
## What is Server Manager?

**Server Manager** is a management console available in Windows Server 2012. It is used to manage and configure server roles and features, helping IT professionals manage both local and remote Windows-based servers.

### Features of Server Manager

1. **Manage Multiple Servers**:

   * Server Manager allows you to manage up to 100 servers from a single console. You can add remote servers and view their status.

2. **Add Roles and Features**:

   * You can use Server Manager to add various roles and features to the server, such as Web Server (IIS), File Server, and Active Directory Domain Services.

3. **Monitor Server Performance**:

   * Server Manager provides the ability to monitor server performance, allowing you to track the health and performance of the server.

4. **Configure Event Data**:

   * You can configure event data settings to limit the amount of data received from the server.

5. **Best Practices Analyzer (BPA)**:

   * By using BPA, you can review the server's settings and ensure they align with best practices for the server.

### How to Use Server Manager

#### Opening Server Manager

1. **From Command Line**:

   * Press Windows + R, type "ServerManager," and press Enter.

2. **From Taskbar**:

   * Click on the Server Manager icon, which is by default located on the taskbar.

3. **From Start Screen**:

   * Click the Start button and search for "Server Manager."

#### Adding Servers

1. **Open Server Manager**.
2. **Click on "Add Servers"**.
3. **Type the server name** or click "Browse" to search for a server.
4. **Click "OK"**.

#### Adding Roles and Features

1. **Go to the "Dashboard" in Server Manager**.
2. **Click on the "Add roles and features" link**.
3. **Click "Next" and choose "Role-based or feature-based installation"**.
4. **Select the server** and click "Next".
5. **Choose the roles and features**, then click "Next".
6. **Confirm the installation** and click "Install".

### Conclusion

**Server Manager** is a powerful management tool in Windows Server 2012 that simplifies and streamlines server administration. It allows you to manage multiple servers from a single location, add roles and features, and monitor server performance.
