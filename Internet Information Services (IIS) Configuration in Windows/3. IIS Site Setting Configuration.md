### **IIS Manager All Site Settings**

---

### **1. IIS Manager Overview**

IIS Manager, or Internet Information Services Manager, is a graphical interface used to manage websites in Windows Server. To open it, go to the **Control Panel**, then **System and Security**, and finally click on **Internet Information Services (IIS) Manager** under **Administrative Tools**.

---

### **2. Sites Section**

* **Sites**: In IIS Manager, in the left-hand **Connections** pane, click on **Sites**. Here, you can see a list of all hosted websites.
* **Default Website**: This is the website created by default by IIS. You can edit or remove it as per your needs.

---

### **3. Site Bindings**

* **Site Bindings**: To edit the binding settings of a website, right-click on your website and choose **Edit Bindings**.
* **Bindings Dialog Box**: Here, you will have the option to add or edit bindings.

#### **Binding Settings**:

* **Type**: Choose the protocol (HTTP or HTTPS).
* **IP Address**: If you want to bind to a specific network card, select it. "All Unassigned" means any connection will be accepted.
* **Port**: The default port for HTTP is 80, and for HTTPS, it is 443.
* **Host Name**: This field is typically left empty. If you fill it in, it will accept requests only for that specific host name.

---

### **4. Adding a New Binding**

* **Add Button**: In the Site Bindings window, click the **Add** button.
* **Enter Binding Information**: Fill in the required information and click **OK**.

---

### **5. Application Pool Configuration**

* **Application Pools**: In IIS, each website has an application pool that isolates its processes. To manage application pools, click on **Application Pools** in IIS Manager.
* **Start/Stop/Recycle**: Here, you can start, stop, or recycle the application pool.

---

### **6. Authentication Settings**

* **Authentication**: To manage the security settings of your website, right-click on the website and choose **Authentication**. Here, you can set options like Windows Authentication, Basic Authentication, etc.

---

### **7. SSL Configuration for HTTPS**

To set up HTTPS, you will need an SSL certificate:

1. In IIS Manager, click on your server name.
2. Double-click on **Server Certificates**.
3. Choose **Create Self-Signed Certificate** and fill in the necessary information.

---

### **8. URL Rewrite Module**

Using URL Rewrite, you can manage the URL structure, which is important for SEO:

1. Right-click on the website and choose **URL Rewrite**.
2. Here, you can create new rules or edit existing ones.

---

### **9. FTP Site Configuration (Optional)**

If you want to set up an FTP site:

1. Right-click on **Sites** in IIS Manager and choose **Add FTP Site...**.
2. Specify the FTP site name and physical path.
3. Set binding settings and authentication options.

---
