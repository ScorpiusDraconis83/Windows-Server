
## You can run the following commands via the Run dialog (Win + R) or in the Command Prompt/terminal:

| No. | Command         | Description                           | Usage (What You Can Do)                                                 |
|-----|-----------------|---------------------------------------|--------------------------------------------------------------------------|
| 1   | msconfig        | System Settings.                      | Boot settings, startup control, services management.                    |
| 2   | msinfo32        | System Information.                   | View detailed system summary and hardware resources.                    |
| 3   | resmon          | Resource Monitor.                     | Monitor CPU, memory, disk, and network usage.                           |
| 4   | main.cpl        | Mouse Settings.                       | Change mouse pointers, buttons, speed settings.                         |
| 5   | mstsc           | Remote Desktop Service.               | Connect remotely to another computer.                                   |
| 6   | cmd             | The Command Prompt.                   | Execute command-line tools and scripts.                                 |
| 7   | explorer        | Windows Explorer.                     | Open file explorer to browse files and folders.                         |
| 8   | taskmgr         | Task Manager.                         | View and manage running processes, apps, and system performance.        |
| 9   | shutdown        | Windows Shutdown.                     | Shutdown, restart, or log off the computer.                             |
| 10  | chkdsk          | Check Disk Utility.                   | Scan and fix disk errors.                                               |
| 11  | cleanmgr        | Clean Disk Manager.                   | Delete unnecessary files to free up space.                              |
| 12  | dxdiag          | DirectX Options.                      | View DirectX info and troubleshoot graphics/sound problems.             |
| 13  | powershell      | Windows PowerShell Console.           | Run advanced scripts and system administration tasks.                   |
| 14  | winver          | Windows Version.                      | Display current Windows version and build.                              |
| 15  | control folders | Folder Options.                       | Customize folder view and behavior.                                     |
| 16  | diskmgmt.msc    | Disk Manager.                         | Create, delete, and format disk partitions.                             |
| 17  | eventvwr.msc    | Event Viewer.                         | View logs of system events, errors, and warnings.                       |
| 18  | gpedit.msc      | Local Group Policy Editor.            | Configure system and user policy settings.                              |
| 19  | regedit         | Registry Editor.                      | Edit Windows registry keys and values.                                  |
| 20  | sysdm.cpl       | System Properties.                    | Manage system performance, name, remote access, etc.                    |
| 21  | powercfg.cpl    | Power Options.                        | Change power plans and sleep settings.                                  |
| 22  | magnify         | Magnifier.                            | Launch screen magnifier for better visibility.                          |
| 23  | charmap         | Windows Character Table.              | Select and copy special characters/symbols.                             |
| 24  | ncpa.cpl        | Network Connections.                  | Enable/disable network adapters and view connection status.             |
| 25  | mrt             | Malware Removal Tool.                 | Scan and remove common malicious software.                              |
| 26  | devmgmt.msc     | Device Manager.                       | Manage hardware devices and drivers.                                    |
| 27  | netplwiz        | User accounts.                        | Add/remove user accounts and change login settings.                     |
| 28  | services.msc    | Services.                             | Start/stop/configure Windows background services.                       |
| 29  | appwiz.cpl      | Programs and Components.              | Uninstall or change installed programs.                                 |
| 30  | control         | Control Panel.                        | Access various system configuration tools.                              |
| 31  | “.”             | Open the folder of the current user.  | Opens the current user's home directory.                                |
| 32  | osk             | On-Screen Keyboard.                   | Opens virtual keyboard on the screen.                                   |
| 33  | snippingtool    | Scissors (Snipping Tool).             | Capture screenshots of part or whole screen.                            |
| 34  | mdsched         | Windows Memory Checker.               | Test computer memory for errors.                                        |
