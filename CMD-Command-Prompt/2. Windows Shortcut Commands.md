## You can run the following commands via the Run dialog (Win + R) or in the Command Prompt/terminal:


| No. | Command         | Description                           |
|-----|-----------------|---------------------------------------|
| 1   | msconfig        | System Settings.                      |
| 2   | msinfo32        | System Information.                   |
| 3   | resmon          | Resource Monitor.                     |
| 4   | main.cpl        | Mouse Settings.                       |
| 5   | mstsc           | Remote Desktop Service.               |
| 6   | cmd             | The Command Prompt.                   |
| 7   | explorer        | Windows Explorer.                     |
| 8   | taskmgr         | Task Manager.                         |
| 9   | shutdown        | Windows Shutdown.                     |
| 10  | chkdsk          | Check Disk Utility.                   |
| 11  | cleanmgr        | Clean Disk Manager.                   |
| 12  | dxdiag          | DirectX Options.                      |
| 13  | powershell      | Windows PowerShell Console.           |
| 14  | winver          | Windows Version.                      |
| 15  | control folders | Folder Options.                       |
| 16  | diskmgmt.msc    | Disk Manager.                         |
| 17  | eventvwr.msc    | Event Viewer.                         |
| 18  | gpedit.msc      | Local Group Policy Editor.            |
| 19  | regedit         | Registry Editor.                      |
| 20  | sysdm.cpl       | System Properties.                    |
| 21  | powercfg.cpl    | Power Options.                        |
| 22  | magnify         | Magnifier.                            |
| 23  | charmap         | Windows Character Table.              |
| 24  | ncpa.cpl        | Network Connections.                  |
| 25  | mrt             | Malware Removal Tool.                 |
| 26  | devmgmt.msc     | Device Manager.                       |
| 27  | netplwiz        | User accounts.                        |
| 28  | services.msc    | Services.                             |
| 29  | appwiz.cpl      | Programs and Components.              |
| 30  | control         | Control Panel.                        |
| 31  | “.”             | Open the folder of the current user.  |
| 32  | osk             | On-Screen Keyboard.                   |
| 33  | snippingtool    | Scissors (Snipping Tool).             |
| 34  | mdsched         | Windows Memory Checker.               |
