
## 🌐 **Definition of NAT and ARP (English + Short + ELI10 Style)**

---

## 🧱 **NAT (Network Address Translation)**

> **NAT** is a technique that **converts a Private IP into a Public IP** so your device can connect to the internet.


---

### 📌 Why is NAT important?

* Public IPs are limited
* It lets multiple devices use **one internet connection**

---

## 📡 **ARP (Address Resolution Protocol)**

> **ARP** is a protocol that **converts an IP address into a MAC address** — so data can reach the correct device in a local network.

### 📌 When does ARP work?

* When one device wants to send a packet to another within the same local network
* It gets the MAC address from the IP and delivers the data

---

## ⚔️ **NAT vs ARP – Super Simple Comparison**

| 🔍 Feature         | 🌐 NAT                       | 📡 ARP                      |
| ------------------ | ---------------------------- | --------------------------- |
| Full Form          | Network Address Translation  | Address Resolution Protocol |
| What does it do?   | Converts Private ↔ Public IP | Finds MAC from IP           |
| Where is it used?  | On Gateway or Router         | Inside Local Network (LAN)  |
| Translates between | IP ↔ IP                      | IP ↔ MAC                    |
