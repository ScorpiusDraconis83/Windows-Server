## 1. Netting

**Netting** is a networking process used to manage IP addresses. This technique is particularly important when multiple devices in a private network are connected to the internet through a single public IP address.

### **Key Points:**

* **NAT (Network Address Translation)**: This technique converts private IP addresses into public IP addresses.
* **Security**: NAT provides security by hiding private IP addresses from the internet.

## 2. ARP (Address Resolution Protocol)

**ARP** is a communication protocol used to convert an IP address into a MAC address.

### **ARP -a Command**

* **ARP Cache**: The `arp -a` command can be used to view the ARP cache, which contains all known IP addresses and their corresponding MAC addresses.
* **Wireshark**: Wireshark is a network analysis tool that can show information about ARP packets. It displays only MAC addresses; IP addresses are not shown.

### **How ARP Works:**

1. When a device wants to contact another device, it first checks the ARP cache to see if the MAC address is already available.
2. If the MAC address is not found, the device sends an ARP request that is broadcasted to all devices in the LAN.
3. The device with the matching IP address responds with its MAC address.

## 3. NAT (Network Address Translation)

**NAT** is a technique that converts a client’s (user's) private IP address into a public IP address provided by the ISP (Internet Service Provider).

### **Key Points:**

* **Role of Router**: A router has two interfaces: LAN (Local Area Network) and WAN (Wide Area Network).

  * The LAN interface has private IP addresses for devices.
  * The WAN interface has public IP addresses.
* **Data Transmission**: The router manages data transmission, sending requests both from inside to outside and outside to inside.

## 4. Port Forwarding

**Port Forwarding** occurs when a request from an external network is directed toward an internal network.

### **Key Points:**

* It is necessary when a client is outside the network and needs to access an internal server or device.

## 5. Switch and Proxy

### **Switch**

* A switch manages data traffic within a LAN using MAC addresses. It ensures that data reaches the correct device.

### **Proxy**

* A proxy server stores all information, allowing for control over data traffic.

## Conclusion

All these techniques are used to accomplish various tasks in networking. NAT and ARP are both important, as NAT helps connect devices to the internet, while ARP establishes communication between devices on a local network.
