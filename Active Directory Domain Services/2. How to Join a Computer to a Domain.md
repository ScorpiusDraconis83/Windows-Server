
# How to Join a Computer to a Domain
The computer to an Active Directory network, allowing it to use domain credentials for login, access domain resources, and apply domain policies for security and management.

## Step 1: System Setup  
- **Set up the system**:  
  - Make sure your computer is connected to the network. For example, if you’re using an office network, ensure your computer is connected via Wi-Fi or Ethernet.  
  - The IP address should be correct. You can check your IP address by typing `ipconfig` in the command prompt.

---

## Step 2: Open System Properties  
- **Go to Properties**:  
  - Right-click on "This PC" and select "Properties".  
  - Here you will see system details like Windows version, processor, RAM, etc.

- **Advanced System Settings**:  
  - Click on "Advanced system settings" on the left side.  
  - This will take you to the System Properties window.

---

## Step 3: Change Computer Name  
- **Computer Name Tab**:  
  - Go to the "Computer Name" tab. You will see your computer’s current name.  
  - Click the "Change" button.

- **Join a Domain**:  
  - In the "Member of" section, select the "Domain" option.  
  - Enter your domain name here. For example, if your domain is `example.local`, type that in.

---

## Step 4: Authentication  
- **Username and Password**:  
  - When you enter the domain name, you will be asked to provide a username and password.  
  - For example, if your domain administrator username is `admin` and the password is `password123`, fill those in correctly.

---

## Step 5: Restart the Computer  
- **Restart the Computer**:  
  - If everything is correct, you’ll get a message saying that your computer has successfully joined the domain.  
  - You’ll be prompted to restart. Click "OK", then "Close", and restart your computer.

---

## Step 6: Log In  
- **Log in with the Domain**:  
  - After restarting, choose the "Other user" option on the login screen.  
  - Now log in using your domain username and password.  
    - Example: If your username is `user1@example.local` and the password is `password123`, enter those.


---
