
## Introduction to Port Forwarding

Port forwarding, or port mapping, is a process of network address translation (NAT) that redirects communication requests from a specific address and port number to another address and port. This technique is primarily used in routers and firewalls, enabling access from external networks to internal machines.

## **Types of Port Forwarding**

1. **Local Port Forwarding**

   * This process is used to send data from a local machine to a remote server using SSH (Secure Shell). For example, if you want to access an FTP server from your home computer, you can map a local port to that server's FTP port.

2. **Remote Port Forwarding**

   * In this type, access to services on an internal network from a remote machine is allowed. For example, if an employee wants to make an FTP server available in the company network from their home, they can use remote port forwarding.

3. **Dynamic Port Forwarding**

   * This technique works through a SOCKS proxy, allowing clients to securely connect to a trusted server. It is commonly used in situations where multiple services need to be accessed simultaneously.

## **How Port Forwarding Works**

### **Basic Example**

Suppose you have a CCTV camera located on your local network, and you want to access it from the internet. The following steps would be required:

1. **Router Configuration:**

   * Go to your router's settings and find the "Port Forwarding" option.
   * Add a new rule where:

     * **Service Name:** CCTV
     * **External Port:** 5000 (which will be accessed over the internet)
     * **Internal IP Address:** 192.168.1.10 (the local IP of your CCTV)
     * **Internal Port:** 5000

2. **Firewall Settings:**

   * Ensure that your router’s firewall allows traffic on this port.

3. **Testing the Connection:**

   * Now, you can access your CCTV camera from your mobile or any other device by going online. You just need to use your public IP address and port 5000.

## **Security Considerations**

When performing port forwarding, it is crucial to keep security in mind. To prevent unauthorized users, consider the following precautions:

* Use strong passwords.
* Only open necessary ports.
* Regularly review your router and firewall settings.
