
## What is a Computer Proxy?

A **computer proxy** is an intermediary server that sits between a user's device and the internet. It forwards requests from the device to the internet and then sends the response back to the device. Proxies are used to improve privacy, security, and performance by hiding the user's IP address, filtering content, and caching data for faster access.


### Functions of Proxy Servers

1. Request Management
2. Data Caching
3. Security and Privacy
4. Filtering and Content Scanning

### Types of Proxies

1. Forward Proxy
2. Reverse Proxy
3. Anonymous (Transparent) Proxy

### Benefits of Using Proxies

* Privacy
* Security
* Performance Improvement
* Content Access

### How a Proxy Works in an ISP Medium Hub PC or Client

To understand the basics of a proxy server and its configuration, here's a step-by-step overview of how a proxy is set up and used in an ISP medium hub setup:

#### **1. Understanding Proxy Servers**

* **Definition**: A proxy server acts as an intermediary between your device (PC or client) and the internet. It forwards requests from the client to the servers and sends the server's response back to the client.
* **Purpose**: Proxies are used to enhance security, improve privacy, and bypass content restrictions.

#### **2. Setting Up a Proxy in a Medium Hub Environment**

1. **Identify the Proxy Server**:

   * Obtain the proxy server’s IP address and port number from your network administrator or ISP.

2. **Configure Proxy Settings on the Client PC**:

   * On Windows 10/11:

     * Go to **Settings** > **Network & Internet** > **Proxy**.
     * Under **Manual proxy setup**, turn on **Use a proxy server**.
     * Enter the **Address** (IP of the proxy server) and **Port** (typically 8080 or 3128).
     * Click **Save**.

3. **Use an Automatic Configuration Script (if available)**:

   * If your organization provides a PAC (Proxy Auto-Configuration) file, you can enable it. Set **Use setup script** to **On** and enter the script URL.

4. **Test the Connectivity**:

   * After configuring, open a web browser and try accessing a website to ensure the proxy settings are working correctly.

#### **3. How It Works in Practice**

* When you make a request (such as visiting a website), the request does not go directly to the internet; it first goes to the proxy server.
* The proxy server processes this request, retrieves data from the target website, and sends it back to your PC.
* This process helps to hide your IP address, increasing privacy and improving load times through caching.

#### **4. Benefits of Using a Proxy in an ISP Medium Hub**

* **Increased Privacy**: Your real IP address is hidden from websites.
* **Improved Security**: Proxies can filter dangerous traffic and provide additional security against cyber threats.
* **Access Control**: Organizations can restrict access to certain websites or types of content.
* **Performance Improvement**: Frequently accessed content can be cached by the proxy, reducing load times for users.

#### **5. Troubleshooting Common Issues**

* If you face issues while using the proxy:

  * Ensure the proxy settings are entered correctly.
  * Check if the proxy server is running.
  * Make sure firewall settings are not blocking access.
  * Test connectivity without the proxy to rule out other network issues.

By following these steps, you can effectively set up and use a proxy in an ISP medium hub environment, improving your browsing experience while maintaining security and privacy.
