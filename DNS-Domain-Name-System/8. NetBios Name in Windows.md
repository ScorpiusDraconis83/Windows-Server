### **What is a NetBIOS Name?**

A NetBIOS name is a name for a computer that helps identify it on a network.

#### **Example**

Let’s say your computer's name is "RAM-PC."

* **NetBIOS name**: "RAM-PC"

This name tells other computers on the network which computer it is.

#### **Important Points**

1. **Length**: The NetBIOS name can be up to 15 characters long.
2. **Special Characters**: It can include letters, numbers, and some special characters like periods (.), but not spaces.
3. **16th Character**: The 16th character is reserved for a special code that indicates what the computer does.

#### **How Does It Work?**

When you connect to another computer on your network, the NetBIOS name helps identify that computer.

#### **Conclusion**

The NetBIOS name is an identity that distinguishes your computer from others on the network.
