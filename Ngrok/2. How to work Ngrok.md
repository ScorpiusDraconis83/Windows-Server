## How to Work with Ngrok

Ngrok is a simple and effective tool that allows you to access your local server on the internet. In this guide, we will discuss in detail how to use Ngrok.

## **Step 1: Install Ngrok**

Installing Ngrok is very easy. You can download it for different platforms:

1. **Windows:**

   * Visit the Ngrok website and download the binary for Windows.
   * Unzip the downloaded file and place it in any directory in your PATH.

2. **Linux/Mac:**

   * Run the following command in the terminal:

     ```bash
     curl -s https://ngrok-agent.s3.amazonaws.com/ngrok.asc | sudo gpg --dearmor -o /etc/apt/keyrings/ngrok.gpg
     echo "deb [signed-by=/etc/apt/keyrings/ngrok.gpg] https://ngrok-agent.s3.amazonaws.com buster main" | sudo tee /etc/apt/sources.list.d/ngrok.list
     sudo apt update && sudo apt install ngrok
     ```

## **Step 2: Connect Your Account**

To use Ngrok, you need to create an account. After creating an account, you will receive an **authtoken**. Add it to your terminal:

```bash
ngrok config add-authtoken <YOUR_AUTHTOKEN>
```

## **Step 3: Start a Tunnel**

Now, you can start a tunnel to access your local server on the internet. If your server is running on `http://localhost:8080`, run the following command in the terminal:

```bash
ngrok http 8080
```

This will give you a public URL, such as `https://123456789.ngrok.io`. You can share this URL with anyone so they can access your local server.

## **Step 4: Monitor Your Tunnel**

While Ngrok is running, you can see the log of all requests in the terminal. This helps you understand which requests are coming and what their responses are.

## **Step 5: Secure Your Application**

If you don't want anyone to access your application, you can take security measures. For example, you can add basic authentication:

```bash
ngrok http 8080 --auth "username:password"
```

Or you can use OAuth.

## **Step 6: Running Ngrok Persistently**

If you want Ngrok to always run, you can run it as a service. On Linux, you can use systemd, while on Windows, you can set it up as a Windows service.

## **Conclusion**

Ngrok is a very useful tool that allows developers to easily share their local server. With it, you can share your projects with others without any complex setup. Whether you're working on web development or API testing, Ngrok makes your work much easier.

#### How to Use Ngrok on PC

If you want to use Ngrok on your PC, here's a detailed step-by-step guide.

##### Step 1: Download Ngrok

1. **Go to the Ngrok website**: Visit [ngrok.com](https://ngrok.com).
2. **Sign up**: If you don't have an account, sign up and create one.
3. **Download Ngrok**: After signing up, go to the download section and download the Windows version of Ngrok.

##### Step 2: Install Ngrok

1. **Unzip the ZIP file**: Find the downloaded ZIP file and unzip it.
2. **Place Ngrok in a convenient location**: Place the `ngrok.exe` file in a folder where you keep your executable files, or you can keep it in your Downloads folder.

##### Step 3: Connect Your Ngrok Account

1. **Open Command Prompt**: Press `Win + R`, type `cmd`, and press Enter.
2. **Navigate to Ngrok's location**: Change the directory to the location of `ngrok.exe`. For example, if it’s in the Downloads folder:

   ```
   cd C:\Users\YourUsername\Downloads
   ```
3. **Authenticate with the Auth Token**:

   * Go to your Ngrok dashboard and copy your Auth Token.
   * In the command prompt, type the following command:

   ```
   ngrok authtoken YOUR_AUTH_TOKEN
   ```

   Replace `YOUR_AUTH_TOKEN` with the token you copied.

##### Step 4: Start Ngrok

1. **Run your local server**: Ensure your local server (such as a web application) is running. For example, if you're using port 5000 for the local server, keep it running.
2. **Start Ngrok**: In the command prompt, run the following command:

   ```
   ngrok http 5000
   ```

   Here, replace `5000` with the port number your local server is using.

##### Step 5: Access the Public URL

1. **View Ngrok output**: After running the command, Ngrok will display a forwarding URL, such as `http://abcd1234.ngrok.io`.
2. **Share the URL**: You can now share this URL with anyone who needs to access your local server.

#### Conclusion

By following these steps, you can easily set up and use Ngrok on your PC. It’s a great tool for developers who want to expose their local applications for testing or sharing purposes. Enjoy the flexibility that Ngrok offers!
