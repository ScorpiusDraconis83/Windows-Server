
# 🌐 How to Use Ngrok – Step-by-Step Guide

Ngrok is a powerful tool that creates secure tunnels to your local server, making it accessible over the internet. It is especially useful for developers testing webhooks, sharing work, or testing APIs on local environments.

> 🔗 **Official Website**: [https://ngrok.com](https://ngrok.com)

---

## ✅ Step 1: Sign Up and Download Ngrok

1. Go to the official website: [https://ngrok.com](https://ngrok.com)
2. Create a free account.
3. After signing in, go to the **[Download page](https://ngrok.com/download)**.
4. Download the version based on your OS (Windows, Linux, macOS, etc.).

---

## ✅ Step 2: Install Ngrok

### 🔹 For **Windows**:

1. Extract the downloaded `.zip` file.
2. Move `ngrok.exe` to a convenient folder (e.g., `C:\ngrok`).
3. (Optional) Add the folder path to your system **Environment Variables** for easier access via CMD.

### 🔹 For **Linux/Debian-based systems** (like Ubuntu/Kali):

```bash
curl -s https://ngrok-agent.s3.amazonaws.com/ngrok.asc | gpg --dearmor -o /etc/apt/keyrings/ngrok.gpg
echo "deb [signed-by=/etc/apt/keyrings/ngrok.gpg] https://ngrok-agent.s3.amazonaws.com stable main" | tee /etc/apt/sources.list.d/ngrok.list
apt update && apt install ngrok
```

---

## ✅ Step 3: Connect Your Ngrok Account

1. Log into your Ngrok dashboard: [https://dashboard.ngrok.com/get-started/setup](https://dashboard.ngrok.com/get-started/setup)
2. Copy your **Auth Token**.
3. Run the command:

```bash
ngrok config add-authtoken <YOUR_AUTH_TOKEN>
```

Replace `<YOUR_AUTH_TOKEN>` with your actual token.

---

## ✅ Step 4: Run Your Local Server

Start the service or server you want to expose. For example:

```bash
python3 -m http.server 8080
```

This starts a server on `localhost:8080`.

---

## ✅ Step 5: Start Ngrok Tunnel

Expose your local port using:

```bash
ngrok http 8080
```

Replace `8080` with your local server's port.

Ngrok will now generate a public URL like:

```
https://abc123.ngrok.io
```

You can share this URL to access your local server over the internet.

---

## ✅ Step 6: Optional Features

### 🔐 Basic Authentication

Add simple authentication to protect your tunnel:

```bash
ngrok http 8080 --auth="username:password"
```

### 📈 Web Interface

Monitor incoming requests using Ngrok’s local web interface:

```
http://127.0.0.1:4040
```

---

## ✅ Bonus: Run Ngrok on Boot (Linux only)

You can create a systemd service to auto-start Ngrok if needed.

---

## ✅ Final Thoughts

Ngrok simplifies exposing local servers securely. Whether you're testing webhooks, APIs, or sharing projects, it saves time and avoids complicated port forwarding or hosting setups.
