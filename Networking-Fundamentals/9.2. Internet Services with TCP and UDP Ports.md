
## ✅ Internet Services with TCP/UDP Ports – Use Case Based Table

| Priority | Service Name   | Port No.  | Protocol | Use Case / Purpose                             | Prefer Why?                          |
| -------- | -------------- | --------- | -------- | ---------------------------------------------- | ------------------------------------ |
| 1        | HTTP           | 80        | TCP      | Website browsing (non-secure)                  | TCP ensures complete web page load   |
| 2        | HTTPS          | 443       | TCP      | Secure website browsing                        | Encrypted + reliable transfer        |
| 3        | DNS            | 53        | UDP/TCP  | Domain name to IP conversion                   | UDP for speed, TCP if response large |
| 4        | FTP            | 21, 20    | TCP      | File Transfer (upload/download)                | Reliable data transmission           |
| 5        | SSH            | 22        | TCP      | Secure remote login (Linux server access etc.) | Secure + Reliable                    |
| 6        | Telnet         | 23        | TCP      | Remote terminal (older protocol)               | Needs accurate command transmission  |
| 7        | SMTP           | 25        | TCP      | Sending Emails                                 | Reliable delivery needed             |
| 8        | IMAP           | 143       | TCP      | Receiving emails (with sync)                   | Reliability for complete email sync  |
| 9        | POP3           | 110       | TCP      | Downloading emails (without sync)              | Reliable email transfer              |
| 10       | DHCP           | 67/68     | UDP      | Assigning IP addresses dynamically             | Fast & connectionless needed         |
| 11       | SNMP           | 161       | UDP      | Network Monitoring/Device Management           | Lightweight, fast                    |
| 12       | NTP            | 123       | UDP      | Network Time Sync                              | Very low delay required              |
| 13       | RDP            | 3389      | TCP      | Remote Desktop to Windows system               | Needs stability                      |
| 14       | MySQL Database | 3306      | TCP      | Database queries                               | Data integrity critical              |
| 15       | PostgreSQL     | 5432      | TCP      | PostgreSQL database service                    | Structured and safe transmission     |
| 16       | SIP (VoIP)     | 5060      | UDP/TCP  | Voice Over IP calling                          | UDP for speed, TCP for NAT traversal |
| 17       | TFTP           | 69        | UDP      | Lightweight file transfer                      | Fast but not reliable                |
| 18       | BitTorrent     | 6881–6889 | TCP/UDP  | Peer-to-peer file sharing                      | Uses both protocols dynamically      |
| 19       | BGP            | 179       | TCP      | Internet Routing Protocol                      | Stability and correctness needed     |
| 20       | LDAP           | 389       | TCP/UDP  | Directory services like Active Directory       | TCP for search, UDP for discovery    |
| 21       | NetBIOS        | 137–139   | UDP/TCP  | Windows File Sharing                           | Depends on function used             |
