
## 🧠 **Introduction to Computers**

A **computer** is an electronic machine that takes input, processes it according to defined instructions, and delivers output. It works under the direction of software and user commands.

---

## 🖥️ **Core Components of a Computer**

### 🧱 **Hardware**

Hardware refers to the **physical parts** of a computer that you can touch and see.

**Examples include:**

* **CPU (Central Processing Unit)**
* **Monitor**
* **Keyboard**
* **Mouse**

---

### 💾 **Software**

Software represents the **intangible programs and data** that direct the hardware on what tasks to perform.

**Examples:**

* **Operating Systems** like Windows, Linux
* **Applications** like VLC, MS Paint
* **System Tools & Utilities**

---

## 💡 **Types of Software**

### 1️⃣ **System Software**

Manages and operates the core functions of hardware and other software.

**Examples:**

* Operating Systems (Windows, Linux)
* BIOS (Basic Input/Output System)

---

### 2️⃣ **Application Software**

Built for end-users to perform specific activities.

**Examples:**

* VLC Media Player
* Microsoft Paint
* Word Processors like MS Word

---

### 3️⃣ **Utility Software**

These are tools that support system maintenance, performance improvement, and protection.

**Examples:**

* Antivirus Programs
* Firewalls
* Disk Cleanup Utilities

---

## 🎮 **Input Devices**

Input devices allow users to **send data and commands** to the computer.

**Common examples:**

* Keyboard
* Mouse
* Microphone
* Biometric scanner (e.g., thumbprint scanner)

---

## 🧠 **Operating System (OS)**

An **Operating System** is the bridge between the user and the computer hardware. It manages both the system's resources and user interactions.

---

### ⚙️ **OS Interaction Flow**

* **Windows:** User → Application → OS → Hardware
* **Linux:** User → Shell (CLI) → Kernel → Hardware

---

### 🔧 **Key Functions of an Operating System**

#### 📌 **1. Process Management**

* Schedules tasks and manages running programs
* Controls multitasking and handles process termination

#### 📌 **2. Memory Management**

* Allocates RAM efficiently
* Prevents memory conflicts and optimizes usage

#### 📌 **3. Input/Output (I/O) Management**

* Controls interactions between the system and external devices
* Handles data transfers from peripherals (keyboard, printer, screen, etc.)

---

## 🌐 **Internet vs Intranet**

### 🌍 **Internet**

A **global network** of computers and devices that offers information access, communication, entertainment, and services.

### 🏢 **Intranet**

A **private network** used by an organization for secure internal communication and data sharing. It’s **limited to employees** and not publicly accessible.
