
## Networking Devices in Computer Networks

| Device                                                                                      | Function                                                                                                     | Example                                                                         |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------- |
| [**Hub**](https://www.techopedia.com/definition/2483/hub-networking)                        | Basic device that connects multiple devices and broadcasts data to all of them without filtering.            | Small office network where all computers share resources via a hub.             |
| [**Switch**](https://www.cisco.com/c/en/us/products/switches/what-is-a-network-switch.html) | More intelligent device that sends data only to the device it is intended for, improving network efficiency. | Office LAN connecting computers and printers, directing data precisely.         |
| [**Router**](https://www.cloudflare.com/learning/ddos/glossary/router/)                     | Routes data packets between different networks based on IP addresses.                                        | Home router connects devices to the internet and manages traffic.               |
| [**Bridge**](https://www.geeksforgeeks.org/network-bridge-in-computer-network/)             | Connects two or more network segments and filters traffic based on MAC addresses.                            | Links two LANs in different buildings for communication.                        |
| [**Gateway**](https://www.networkencyclopedia.com/gateway/)                                 | Acts as a gateway between different networks and translates data formats or protocols.                       | Connects company’s internal network to the internet or other external networks. |
| [**Repeater**](https://www.techopedia.com/definition/2537/repeater)                         | Amplifies and regenerates signals to extend transmission range.                                              | Extends Wi-Fi coverage in large buildings or areas with weak signal.            |
| [**Modem**](https://www.techopedia.com/definition/2384/modem)                               | Converts digital data to analog signals and vice versa to communicate over telephone lines.                  | DSL modem connecting a home network to the internet via phone line.             |

## Device Examples, Functions & Images

### 1. Router

**Function:** Routes data between networks using IP addresses.
**Example:** A home router connects all your devices (smartphones, laptops) to the internet and directs data traffic efficiently.
**Image:** ![Router](https://images.unsplash.com/photo-1587202372775-989d05d0b6c0)

### 2. Switch

**Function:** Sends data packets only to the intended recipient device, improving network performance.
**Example:** In an office, a switch connects computers and printers, ensuring data reaches the correct device.
**Image:** ![Switch](https://images.unsplash.com/photo-1581093588401-9e8c4a0b3e5c)

### 3. Hub

**Function:** Broadcasts incoming data to all devices on the network, regardless of the destination.
**Example:** A hub can be used in a small network where all computers receive all data packets.
**Image:** ![Hub](https://images.unsplash.com/photo-1581091870620-3f9c8a1a3d9c)

### 4. Bridge

**Function:** Connects different network segments and filters traffic using MAC addresses.
**Example:** A bridge links two local area networks in separate buildings, allowing them to communicate smoothly.
**Image:** ![Bridge](https://images.unsplash.com/photo-1581091012184-5c4d5f3e3f3c)

### 5. Gateway

**Function:** Translates data between different network protocols or formats.
**Example:** A gateway connects an internal company network to the internet, enabling communication with external networks.
**Image:** ![Gateway](https://images.unsplash.com/photo-1581092334420-3f9c8a1a3d9c)

### 6. Repeater

**Function:** Boosts weak or degraded signals to extend network coverage.
**Example:** Repeaters help Wi-Fi signals reach all corners of a large office building.
**Image:** ![Repeater](https://images.unsplash.com/photo-1581093588401-9e8c4a0b3e5c)

### 7. Modem

**Function:** Converts digital data from a computer into analog signals for telephone transmission, and converts incoming analog signals back to digital.
**Example:** A DSL modem connects your home network to the internet via telephone lines.
**Image:** ![Modem](https://images.unsplash.com/photo-1581091870620-3f9c8a1a3d9c)
